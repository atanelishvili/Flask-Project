# 🏢 REAL ESTATE MANAGEMENT FLASK APPLICATION

## 📝 PROJECT OVERVIEW
მე შევქმენი ეს თანამედროვე სრული ციკლის (Full-Stack) ვებ-აპლიკაცია Flask ფრეიმვორკის გამოყენებით, რათა გავამარტივო უძრავი ქონების მართვა, განცხადებების განთავსება და ძებნა. აპლიკაცია მოიცავს მომხმარებელთა ავტორიზაციას მრავალდონიანი როლებით, ქონების გაფილტვრასა და სტრუქტურირებულ დინამიურ როუტინგს.

იმისათვის, რომ პლატფორმა შევსებულიყო რეალური და დინამიური მონაცემებით ყოველგვარი დამოკიდებულების პრობლემების გარეშე, პროექტში გავაინტეგრირე სტაბილური, ღია საჯარო REST API. აპლიკაციის პირველივე გაშვებისას, სისტემა ავტომატურად ქაჩავს ამ მონაცემებს და შეაქვს ლოკალურ მონაცემთა ბაზაში.

---

## ✨ CORE FEATURES
* 🔄 **დინამიური API ინტეგრაცია**: ავტომატურად მოაქვს და ტვირთავს უძრავი ქონების აქტუალურ განცხადებებს საჯარო რეპოზიტორიიდან პირველივე ჩართვაზე.
* 🔐 **მომხმარებელთა ავტორიზაცია და როლები**: რეგისტრაციისა და სისტემაში შესვლის სრული სისტემა დაჰეშილი პაროლებითა და მიზნობრივი როლებით (*მაგ. მფლობელი, კლიენტი*).
* 🔍 **ქონების ფილტრაცია**: მომხმარებელს შეუძლია განცხადებების გაფილტვრა ქალაქის, საფასო ლიმიტების, გარიგების ტიპის (*ქირავდება/იყიდება*) და ოთახების რაოდენობის მიხედვით.
* 📊 **შედარების მოდული**: ჩაშენებული სისტემა, რომელიც მომხმარებელს საშუალებას აძლევს რამდენიმე შერჩეული ვარიანტი გვერდიგვერდ შეადაროს.
* 🧩 **მოდულური არქიტექტურა**: პროექტი აგებულია სუფთა Flask Blueprint-ების სტრუქტურით, რაც ერთმანეთისგან გამოყოფს ავტორიზაციას (`auth`), განცხადებებსა და შიდა API-ებს.

## 🛠️ ARCHITECTURE & TECH STACK
* 🐍 **ბექენდ ფრეიმვორკი**: Python / Flask
* 🗄️ **მონაცემთა ბაზა & ORM**: SQLite / Flask-SQLAlchemy
* 🌐 **API კომუნიკაცია**: Requests ბიბლიოთეკა მესამე მხარის მონაცემების გამოსათხოვად
* 🎨 **ფრონტენდი**: HTML5, CSS3 (*რესპონსიული და მორგებული კომპონენტები*)

## 🚀 HOW TO RUN THE PROJECT (INSTRUCTIONS FOR EVALUATION)

პროექტის ლოკალურად გასაშვებად და შესამოწმებლად, მიჰყევით ამ მარტივ ნაბიჯებს თქვენს ტერმინალში:

### 1️⃣ დააკლონეთ რეპოზიტორია (CLONE THE REPOSITORY)
```bash
git clone [https://github.com/atanelishvili/Flask-Project.git](https://github.com/atanelishvili/Flask-Project.git)
cd Flask-Project
2️⃣ ჩამოტვირთეთ საჭირო ბიბლიოთეკები (INSTALL REQUIRED DEPENDENCIES)

Bash
pip install flask flask-sqlalchemy requests
3️⃣ გაუშვით აპლიკაცია (RUN THE APPLICATION)
ჩართეთ პროექტის მთავარი ფაილი ლოკალური სერვერის ასამუშავებლად:

Bash
python app.py
4️⃣ გახსენით პროექტი ბრაუზერში (ACCESS THE APPLICATION)
სერვერის წარმატებით ჩართვის შემდეგ, გახსენით ბრაუზერი და გადადით მისამართზე:

[http://127.0.0.1:5000](http://127.0.0.1:5000)
📢 მნიშვნელოვანია: პირველივე გაშვებისას სისტემა თავად შექმნის მონაცემთა ბაზის სტრუქტურას (realestate.db) და ავტომატურად წამოიღებს ახალ მონაცემებს საჯარო API-დან, ასე რომ თქვენ პირდაპირ დაგხვდებათ შევსებული და მზა კატალოგი.
